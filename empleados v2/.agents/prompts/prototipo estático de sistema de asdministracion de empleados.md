# Prototipo UI - Sistema de Administración de Empleados

## Objetivo
Diseñar un prototipo estático (HTML, CSS, JS) para la gestión de empleados basado en requerimientos refinados.

## Restricciones técnicas
- NO usar frameworks (React, Angular, etc.)
- Solo HTML, CSS, JS puro
- Estructura modular por archivos

## Estructura generada
- /prototipo
  - /css
    - main.css (layout base)
    - components.css (componentes UI)
  - /js
    - data.js (datos mock)
    - app.js (lógica general + sidebar)
  - index.html (login)
  - páginas internas (dashboard, empleados, turnos, etc.)

## Decisiones UX/UI
- Sidebar reutilizable generado por JS
- Layout consistente en todas las páginas
- Separación clara entre estilos base y componentes

## Funcionalidades incluidas
- Login
- Navegación por módulos
- Visualización de empleados
- Configuración de turnos

## Pendientes / mejoras
- Validaciones reales
- Integración con backend
- Manejo de estados dinámicos

## Instrucción para Copilot
Usar esta estructura para extender el sistema agregando nuevas vistas o módulos manteniendo consistencia visual y arquitectura.
